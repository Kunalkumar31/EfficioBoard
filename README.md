#  EfficioBoard

EfficioBoard is a modern Kanban Board application built using JavaScript that helps users visualize their work, optimize task flow, and improve overall efficiency.

It provides an intuitive drag-and-drop interface to manage tasks across different stages, making workflow organization simple and effective.

---

## ✨ Features

* 📌 Visualize tasks using Kanban-style columns
* 🔄 Drag and drop tasks between stages
* ➕ Add, edit, and delete tasks
* 💾 Data persistence using Local Storage
* 📱 Responsive and user-friendly UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 🎯 Purpose

EfficioBoard is designed to help users:

* Visualize their work clearly
* Optimize task management process
* Improve productivity and efficiency
